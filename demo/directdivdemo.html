jQuery YouTube playlist plugin

back to blog post

The html code for the following example

    <div class="youtube"></div>
    <ul class="demo1">
        <li><a href="http://www.youtube.com/watch?v=QBBWKvY-VDc">Video 1</a></li>
        <li><a href="http://www.youtube.com/watch?v=ZXMQqLnRhRI">Video 2</a></li>
        <li><a href="http://www.youtube.com/watch?v=Wvo-g_JvURI">Video 3</a></li>
    </ul>

javascript for the following example

    var holder = $('div.youtube');
    $("ul.demo1").ytplaylist({holder: holder});
